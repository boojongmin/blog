### 26.피보나치 시퀀스 [Fibonacci Sequence](https://www.4clojure.com/problem/26)

주제: Fibonacci seqs

`문제`
피보나치 숫자들중 주어진 인자값만큼 피보나치 시퀀스를 반환하는 함수를 작성하세요.

    Write a function which returns the first X fibonacci numbers.

 

`문제 코드`
